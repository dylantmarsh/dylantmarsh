CTO @ Profile, LLC
------------------

I have been professionally authoring software for over a decade. From web applications, graphic design, microservices, and desktop applications, to hybrid mobile, I have industry experience working with nearly every aspect of the software development process. I am currently the Chief Technology Officer at Profile, the leading assessment provider for athletic teams and a company I co-founded, which has been an entrepreneurial dream come true.
